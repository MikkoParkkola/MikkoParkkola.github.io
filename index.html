Open-source tools for AI coding agents and for everyday use. Most ship as a single binary or a one-line install. GitHub profile →

# add the plugin marketplace in Claude Code
claude plugin marketplace add MikkoParkkola/claude-marketplace
claude plugin install glyphdown # or: nab, anti-ai-tell, trvl

For AI agents

nabRust
Fetch any URL as clean markdown for LLM agents. Real browser-cookie auth, anti-bot reach, prompt-injection defense on by default, plus on-device speech-to-text and OCR. CLI, MCP server, and Claude Code plugin.
trvlGo
Search flights, hotels, trains, cars, and ferries from your AI assistant. No API keys, one binary, price alerts built in. Works with Claude, Cursor, Windsurf, and Codex.
glyphdownRust
Cut token cost for Claude Code and coding agents, losslessly and on-device. Compresses tool output, dedupes context, compacts the system prompt. Stacks on prompt caching.
anti-ai-tellPython
Keep your writing from reading as AI-generated. Prompt constraints, an evidence-based linter, and a judgment checklist. Claude Code plugin with adapters for any client.
mcp-gatewayRust
Run every MCP server behind one port. Meta-MCP routing trims about 95% of the context tokens a full tool list would otherwise cost.
axterminatorRust
Let agents drive macOS. 30 tools for background GUI testing, audio and camera capture, and virtual desktops.

Everyday, no AI needed

nowifiGo
Stuck behind a captive portal? One command, 27 techniques to get back online.
translateTypeScript
Translate the content other tools miss: text that loads and updates dynamically, the kind Google Translate and built-in browser translators leave untouched. Ten-plus providers, with optional on-device models so nothing leaves your machine.
Install via Homebrew: brew tap MikkoParkkola/tap. Source for every project lives on GitHub.
